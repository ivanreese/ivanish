Diminished Fifth

This is the fifth project in the A Shrinking Feeling lineage. Like most of the previous entries in the series, this is a procedural music generation program intended for live exhibition. It's written in ClojureScript — badly — and runs in the browser using the Web Audio API. The code is up on HitBug.

Click the image to view the project. Currently does not work on phones/tablets (soon). It takes a minute to load (8MB-ish). You'll know it's ready when the background turns grey and you see the word "Orchestra" in the top left. Then click the Play button to begin. There's no interactivity — just enjoy the music. Hit Pause if you need to, and then hit Play again to resume. Tick Once is for debug and you can ignore it. Fullscreen does what you'd expect — hit esc on your keyboard to exit. Finally, Sound Check plays a random sample from the system, and is useful when I'm wiring this up for a live performance.
